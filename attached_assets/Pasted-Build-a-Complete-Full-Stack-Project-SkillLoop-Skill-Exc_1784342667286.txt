Build a Complete Full-Stack Project: SkillLoop – Skill Exchange Platform

Create a complete, production-ready full-stack web application named SkillLoop – Skill Exchange Platform for Students.

Technology Stack

- Frontend: React.js with Vite
- Backend: Java Spring Boot (Maven)
- Database: MySQL
- ORM: Spring Data JPA
- API Communication: Axios + REST APIs
- Build Tool: Maven
- Package Manager: npm

Project Description

SkillLoop is a web application where students can exchange skills without monetary transactions. Students can register, log in, add skills they can teach, add skills they want to learn, find matching students, book learning sessions, give ratings, earn credits, and appear on a leaderboard.
